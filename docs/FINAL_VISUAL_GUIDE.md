## 🎯 User Flow

```
1. LAND ON PAGE
   See: Beautiful interface, world map
   Feel: Professional, trustworthy
   
2. SELECT LOCATION
   Action: Click map or enter coordinates
   Feedback: Marker appears, inputs update
   
3. CLICK ANALYZE
   Action: Click button
   Feedback: Loading spinner, "Analyzing..."
   
4. VIEW NDVI MAP
   See: Color-coded vegetation map
   Understand: Red = stressed, Green = healthy
   
5. READ STATISTICS
   See: Mean NDVI, coverage, health %
   Understand: Quantitative data
   
6. READ KEY INSIGHTS
   See: Plain language facts
   Understand: What the data means
   Feel: "Oh, this is serious!"
   
7. READ PREDICTION
   See: 4-step logic chain
   Understand: Why prices will change
   Learn: Economic principles
   
8. SEE RECOMMENDATION
   See: HOLD or SELL
   Understand: What to do NOW
   Feel: Confident in decision
   
9. READ SUMMARY
   See: One powerful sentence
   Remember: Key takeaway
   Act: Make informed decision
```
